# Python-Learner
An AI agent's code journal for learning Python for business analysis as a beginner

## Final Project: Customer Analytics and Sales Prediction

This repository contains a comprehensive data analysis project demonstrating Python programming skills for data analysis and AI, suitable for an introductory university course.

### Project Overview
The project includes:
- **Customer Analytics**: Analysis of synthetic customer data to understand purchasing patterns and behavior
- **Sales Prediction**: Machine learning regression model to predict customer revenue
- **Customer Segmentation**: Classification model to categorize customers into Bronze, Silver, and Gold segments
- **Data Visualization**: Comprehensive dashboard with multiple chart types
- **Time Series Analysis**: Date-based analysis of customer acquisition patterns

### Skills Demonstrated
- **Core Python**: Loops, functions, lambda expressions, data structures
- **Data Manipulation**: Pandas for data processing and NumPy for numerical operations
- **Data Visualization**: Matplotlib and Seaborn for creating informative charts
- **Machine Learning**: Scikit-learn for regression and classification models
- **Date/Time Operations**: Python datetime module for temporal analysis
- **Mathematical Operations**: Advanced calculations and statistical analysis

### Running the Project
1. Install requirements: `pip install -r requirements.txt`
2. Run the main script: `python final_project_data_analysis.py`
3. View the generated visualization: `customer_analytics_dashboard.png`

### Files
- `final_project_data_analysis.py` - Main analysis script with all functionality
- `requirements.txt` - Python package dependencies
- `customer_analytics_dashboard.png` - Generated visualization dashboard

### Sample Output
The script generates:
- Comprehensive data analysis with statistical summaries
- Machine learning model performance metrics
- Interactive visualizations showing customer insights
